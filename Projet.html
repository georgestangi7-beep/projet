Projet : Portfolio Personnel

📌 Contexte

Ce projet a été réalisé dans le cadre de ma formation en BTS SIO option SLAM. L’objectif était de créer un site web personnel afin de présenter mon profil, mes compétences et mes projets réalisés durant ma formation.

🎯 Objectifs

  • Créer un site web moderne et responsive
  • Présenter mes compétences et projets
  • Servir de support pour l’épreuve E4

🛠️ Technologies utilisées

  • HTML5
  • CSS3

💻 Réalisations

  • Création de la structure HTML du site
  • Mise en place du design avec CSS
  • Création des sections : À propos, Compétences, Projets
  • Ajout d’une timeline de parcours
  • Réalisation d’un design responsive

👤 Mon rôle

J’ai réalisé ce projet entièrement en autonomie, de la conception à la mise en œuvre. J’ai conçu le design, développé le site et organisé les contenus.

🧠 Compétences mobilisées

  • Développer une interface web
  • Structurer une page HTML
  • Mettre en forme avec CSS
  • Concevoir une interface utilisateur
